### CS - Software Design Techniques

For designing any application, there are certain ways we can approach them. Below is a list of techniques that I've learned throughout my developer experience.

- - -

| **Technique** | **Explanation** |
| :--- | :--- |
| **Hub and Spoke** | In software architecture, a **hub-and-spoke model** is a pattern where different modules or services (**spokes**) interact through a central controller, API gateway, or orchestrator (**hub**). |
| **Decoupling** | Decoupling means separating components so they do not depend too heavily on each other. Each part can change, upgrade, or be replaced without breaking everything else. |
